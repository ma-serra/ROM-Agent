Objetivo: Transferir e instalar sistema SCEAP Auto Update no Mac
Arquivo: ~/Desktop/sceap_auto_update.zip

================================================================================
MODULO 1: METODOS DE TRANSFERENCIA
================================================================================

OPCAO A: AIRDROP (RECOMENDADO)
1. Abrir Finder
2. Localizar: ~/Desktop/sceap_auto_update.zip
3. Clicar com botao direito > Compartilhar > AirDrop
4. Selecionar Mac de destino

OPCAO B: EMAIL
1. Anexar ~/Desktop/sceap_auto_update.zip em email
2. Enviar para si mesmo
3. Abrir no Mac de destino
4. Baixar anexo

OPCAO C: USB
1. Copiar ~/Desktop/sceap_auto_update.zip para pendrive
2. Conectar pendrive no Mac de destino
3. Copiar arquivo para o Mac

OPCAO D: REDE (MACS NA MESMA REDE)
No Mac de destino:
ifconfig | grep "inet " | grep -v 127.0.0.1

No Mac atual:
scp ~/Desktop/sceap_auto_update.zip usuario@IP_DO_MAC:~/Desktop/

================================================================================
MODULO 2: INSTALACAO NO MAC DE DESTINO
================================================================================

PASSO 1: EXTRAIR ARQUIVOS

cd ~/Desktop
unzip sceap_auto_update.zip

PASSO 2: COPIAR PARA PROJETO SCEAP

cd /caminho/do/seu/sceap
cp -r ~/Desktop/auto_update ./
cp ~/Desktop/test_video_extraction.py ./

PASSO 3: INSTALAR DEPENDENCIAS

Dependencias do sistema de auto-atualizacao:
pip3 install schedule rich click

Dependencias de video (se necessario):
pip3 install openai-whisper ffmpeg-python moviepy pydub

FFmpeg (se necessario):
brew install ffmpeg

PASSO 4: EXECUTAR CORRECAO AUTOMATICA

cd /caminho/do/seu/sceap
python3 auto_update/fix_video_extraction.py

Funcoes do script:
- Configurar SSL
- Baixar modelo Whisper
- Instalar dependencias faltantes
- Verificar FFmpeg

PASSO 5: TESTAR EXTRACAO DE VIDEO

Teste com video:
python3 test_video_extraction.py "/caminho/do/video.mp4"

Diagnostico:
python3 auto_update/diagnose_video_extraction.py

================================================================================
MODULO 3: VERIFICACAO DE FUNCIONAMENTO
================================================================================

COMANDO 1: VERIFICAR ATUALIZACOES

python3 -m sceap.auto_update.cli check -r requirements.txt -v

Saida esperada:
- Sistema atualizado OU
- X atualizacoes disponiveis

COMANDO 2: VALIDAR SISTEMA

python3 -m sceap.auto_update.cli validate -r requirements.txt

Saida esperada:
- Validacao passou: X/X testes
- Taxa de sucesso: 100%

COMANDO 3: TESTAR VIDEO

python3 test_video_extraction.py "/caminho/do/video.mp4"

Saida esperada:
- Extracao de metadados bem-sucedida
- Metadados extraidos
- Confianca: 95%

================================================================================
MODULO 4: CONFIGURACAO OPCIONAL
================================================================================

CRIAR CONFIGURACAO PERSONALIZADA

cd /caminho/do/seu/sceap
cp auto_update/config_template.json auto_update/config.json
nano auto_update/config.json

HABILITAR AGENDAMENTO AUTOMATICO

Verificar status:
python3 -m sceap.auto_update.cli schedule -r requirements.txt --status

Habilitar:
nohup python3 -m sceap.auto_update.auto_update_scheduler \
  --requirements requirements.txt \
  --enable-auto-update > /tmp/sceap_scheduler.log 2>&1 &

Salvar PID:
echo $! > /tmp/sceap_scheduler.pid

PARAR AGENDAMENTO

kill $(cat /tmp/sceap_scheduler.pid)

================================================================================
MODULO 5: COMANDOS UTEIS
================================================================================

VERIFICACOES

Status de todas as ferramentas:
python3 auto_update/diagnose_video_extraction.py

Verificar FFmpeg:
ffmpeg -version
ffprobe -version

Verificar Whisper:
python3 -c "import whisper; print('OK')"

ATUALIZACOES

Atualizar todas as ferramentas:
python3 -m sceap.auto_update.cli update -r requirements.txt -y

Atualizar apenas seguranca:
python3 -m sceap.auto_update.cli update -r requirements.txt --security-only -y

Simulacao (dry-run):
python3 -m sceap.auto_update.cli update -r requirements.txt --dry-run

RELATORIOS

Gerar relatorio completo:
python3 -m sceap.auto_update.cli report -r requirements.txt -o ~/Desktop/relatorio_sceap.md

Ver versao do sistema:
python3 -m sceap.auto_update.cli version

================================================================================
MODULO 6: TROUBLESHOOTING
================================================================================

PROBLEMA: "No module named 'schedule'"
SOLUCAO:
pip3 install schedule rich click

PROBLEMA: "FFmpeg nao encontrado"
SOLUCAO:
brew install ffmpeg
ffmpeg -version

PROBLEMA: "SSL Certificate error"
SOLUCAO:
python3 auto_update/fix_video_extraction.py
pip3 install certifi

PROBLEMA: "Whisper nao carrega modelo"
SOLUCAO:
mkdir -p ~/.cache/whisper
curl -L https://openaipublic.azureedge.net/main/whisper/models/ed3a0b6b1c0edf879ad9b11b1af5a0e6ab5db9205f891f668f8b0e6c6326e34e/base.pt \
  -o ~/.cache/whisper/base.pt

PROBLEMA: Extracao falha
SOLUCAO:
1. Ver logs:
tail -100 /tmp/sceap_autoupdate.log

2. Executar diagnostico:
python3 auto_update/diagnose_video_extraction.py

3. Executar correcao:
python3 auto_update/fix_video_extraction.py

4. Validar sistema:
python3 -m sceap.auto_update.cli validate -r requirements.txt
